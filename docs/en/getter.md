# Get SDK properties
### Get SDK version
Gets the current version of the SDK.

```java
String getSdkVersion();
```

- Return Type : `String`

<details open>
  <summary>Java</summary>

```java
String value = DfineryProperties.getSdkVersion();
```

</details>

<details open>
  <summary>Kotlin</summary>

```kotlin
val value = DfineryProperties.getSdkVersion()
```

</details>
